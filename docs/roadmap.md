# План и дорожная карта

## Поддержка стабильного релиза
- Фиксировать состояние: резервные копии БД и кода перед изменениями.
- Контроль `canAccessPanel()` и флага `is_active` для предотвращения 403 после логина.
- Проверка симлинка `storage` и `public_html` на Beget при любых правках конфигурации.

## Ближайшие шаги
- Улучшить UX импорта Excel: отображение ошибок валидации и подсветка строк.
- Добавить журнал действий по заявкам (кто и когда менял статус/позиции).
- Расширить дашборд метриками по статусам и суммам заказов.
- Повысить покрытие тестами политик и переходов статусов.

## Среднесрочные задачи
- Настроить уведомления (email/мессенджер) по ключевым событиям заявок.
- Добавить историю версий для заявок и позиций.
- Оптимизировать шаблоны заявок: теги, поиск, ограничение доступа по лагерям.

## Технические инициативы
- Ввести CI/CD с автозапуском тестов и линтеров перед деплоем.
- Настроить централизованный сбор логов и оповещения о ошибках.
- Подготовить сценарий восстановления после сбоя (бэкапы MySQL, проверка `storage`).
