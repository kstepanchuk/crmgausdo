# CRM Gausdo

Добро пожаловать в репозиторий **CRM Gausdo**. Здесь собирается вся информация о продукте: цели, ключевые возможности, архитектура и план развития. Репозиторий подготовлен как база знаний для дальнейшей разработки и сопровождения системы.

## О проекте
CRM Gausdo — это корпоративная система для управления взаимоотношениями с клиентами, объединяющая продажи, маркетинг и сервис. Проект ориентирован на быструю настройку под бизнес-процессы, масштабируемость и прозрачную аналитическую отчетность.

## Ключевые возможности
- Единая карточка клиента с историей взаимодействий.
- Воронки продаж с настройкой стадий и автоматизаций.
- Поддержка омниканальных коммуникаций: почта, телефония, мессенджеры.
- Планировщик задач и напоминаний для менеджеров.
- Аналитические отчеты и дашборды.
- Гибкая система прав и ролей.
